Detailed Description

An abstraction of a neural network entity.

Constructor & Destructor Documentation

NNGen.AsyncNeuralNetwork.AsyncNeuralNetwork ( int[]  _neuronCounts,

)
inline

The constructor for a neural network object. Calling this method's write() function will write the entire network.

Parameters
_neuronCountsThe number of neurons in each layer, excluding the bias node.
_biasValuesThe bias value feeding forward into the next layer.
_activationTypesThe activation types for the neurons in each layer
_numIntBitsThe number of integer bits to use for the inputs for each neuron (Sigmoid activated neurons automatically get this set to zero)
_numFracBitsThe number of fractional bits to use for the inputs for each neuron
_numWeightUpperBitsThe number of integer bits used for the weights for each neuron.
_isClassifierIf true, a comparitor will be instantiatated at the output to each node, comparing the output layer nodes to the threshold value.
_classifierThresholdsThe threshold values for the output comparitors
_weightsThe list of weights read in for the neurons from WeightReader.readWeightsFromFile()

<remark> For example, to intialize a classification neural network with three inputs, two hidden sigmoid-activated nodes, and one linear output node, use the following line:</remark>

NeuralNetwork nn = new NeuralNetwork([3, 2, 1], [-1, -1], [..SIGMOID_POLY_APPROX, ..LINEAR], 4, 4, 4, true, [0.5], _weights);
